# Overview

This repository contains a basic EDA project focused on data cleaning and transformation using Python. It demonstrates structured data preprocessing techniques.

## Tech Stack 
- Python
- Pandas
- NumPy


## Tasks Implemented

- Exported a DataFrame to CSV
- Filled missing values
- Transformed the dataset
- Merged different DataFrames into a final consolidated DataFrame
- Created new columns
- Performed groupby and aggregation operations
- Filtered records based on conditions
